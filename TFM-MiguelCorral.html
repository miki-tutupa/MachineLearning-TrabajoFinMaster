4. Entrenamiento del modelo

Particionamiento de los datos

Se realiza un particionamiento de los datos con un 70% para entrenamiento y un 30% para test. El particionamiento se hace de manera estratificada para obtener una proporcin de la variable objetivo similar tanto en el conjunto de entrenamiento como en el de test. Se estandarizan los datos para poder ejecutar el entrenamiento de los diferentes modelos.

In[]:
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import LabelEncoder
from sklearn.preprocessing import StandardScaler

# carga de datos
df = pd.read_csv('creditg_clean.csv')

# codificar variables categricas para evitar error "ValueError: could not convert string to float: '0<=X<200'"
df_encoded = df.copy()
for col in df_encoded.select_dtypes(include=['object']).columns:
    le = LabelEncoder()
    df_encoded[col] = le.fit_transform(df_encoded[col])

# separar datos y clase
data = df_encoded.drop('class', axis=1)
clase = df_encoded['class']

# particionar datos de manera estratificada
data_train, data_test, clase_train, clase_test = train_test_split(data, clase, test_size=0.3, random_state=23, stratify=clase)

# estandarizar datos
standarizer = StandardScaler()
data_train = standarizer.fit_transform(data_train)
data_test = standarizer.transform(data_test)

from scikeras.wrappers import KerasClassifier
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense, Dropout, Input
from tensorflow.keras.optimizers import Adam

# crear un modelo secuencial
def create_model():
    model_nn = Sequential()
    model_nn.add(Input(shape=(data_train.shape[1],)))
    model_nn.add(Dense(64, activation='relu'))
    model_nn.add(Dropout(0.2))
    model_nn.add(Dense(32, activation='relu'))
    model_nn.add(Dense(1, activation='sigmoid'))
    model_nn.compile(optimizer=Adam(learning_rate=0.0001), loss='binary_crossentropy', metrics=['accuracy'])
    return model_nn

# envolver el modelo con KerasClassifier (para evitar error en paso posterior al aadirlo al ensemble)
model_nn_sklearn = KerasClassifier(model=create_model, epochs=100, batch_size=10, verbose=0)

# entrenar el modelo
model_nn_sklearn.fit(data_train, clase_train)

# predicciones
clase_pred_nn_prob = model_nn_sklearn.predict(data_test)
clase_pred_nn = (clase_pred_nn_prob > 0.5).astype(int)

Ensemble final combinando modelos

Intento mejorar los resultados haciendo un ensemble que le pregunte a los diferentes expertos, es decir todos los modelos que hemos entrenado hasta ahora.

In[]:
from sklearn.ensemble import VotingClassifier

# definir los modelos para el ensemble
model_voting = VotingClassifier(estimators=[
    ('rega', model_reg_adjusted),
    ('rf', model_rf),
    ('svm', model_svm),
    ('knn', model_knn),
    ('nn', model_nn_sklearn)
], voting='hard')

# entrenar el ensemble
model_voting.fit(data_train, clase_train)

# predicciones
clase_pred_ensemble = model_voting.predict(data_test)

import pandas as pd
from sklearn.metrics import accuracy_score, f1_score, classification_report

results = []

def evaluate_model(name, clase_true, clase_pred):
    accuracy = accuracy_score(clase_true, clase_pred)
    f1_class_0 = f1_score(clase_true, clase_pred, pos_label=0)
    f1_class_1 = f1_score(clase_true, clase_pred, pos_label=1)
    report = classification_report(clase_true, clase_pred, output_dict=True)
    
    results.append({
        "Modelo": name,
        "Exactitud": accuracy,
        "F1-Score Class 0": f1_class_0,
        "F1-Score Class 1": f1_class_1,
        "Precision Class 0": report['0']['precision'],
        "Precision Class 1": report['1']['precision'],
        "Recall Class 0": report['0']['recall'],
        "Recall Class 1": report['1']['recall']
    })

evaluate_model("Regresin", clase_test, clase_pred_reg_adjusted)
evaluate_model("SVM", clase_test, clase_pred_svm)
evaluate_model("Random Forest", clase_test, clase_pred_rf)
evaluate_model("k-NN", clase_test, clase_pred_knn)
evaluate_model("Red Neuronal", clase_test, clase_pred_nn)
evaluate_model("Ensemble", clase_test, clase_pred_ensemble)

results_df = pd.DataFrame(results)

5. Conclusiones

Despus de entrenar a diferentes modelos, Random Forest aparece como la opcin con mejor desempeo en la variable 1, aunque el ensemble usando todos los modelos combinando y votando las salidas muestra un desempeo prcticamente igual en la variable 1 pero significativamente superior en la variable 0.

En este caso optara por el uso de este ltimo modelo de Ensemble para procesar los datos.
